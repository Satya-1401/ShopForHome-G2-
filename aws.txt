IAM(Identity and access Management, Global Service)
---------------------------------------------------
- Root access created by user by default, shouldnot be used or shared
- Users are people within your organization and can be grouped
- Grps only contains user, not other grps
- Users dont belong to a grp
  and can belong to multiple grps


IAM:permissions
---------------
- Users or grps can be assigned JSON documents aclled policies.
- These policies define the permissions of the users.
- In AWS you can apply LEAST PRIVILAGE PRINCIPLE:dont give more permissions than a user needs.
